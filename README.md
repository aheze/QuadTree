## Implicit Curves

Swift implementation of Jared Hughes's isosurfaces library.

Graph implicit curves!

y * pow(x - y, 2) - (4 * x) - 8 | pow(x, 2) + pow(y, 2) - 5 | tan(pow(x, 2) + pow(y, 2)) - 1
